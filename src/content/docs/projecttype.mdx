Project types are all the addons that add, modify or extends [Minecraft](/minecraft) default features.

To understand each type is necessary to understand which part of the game they affect, where they run and the required software they need to use.

| Project type                   | [Environment](/environment)                                       | [Version](/versions)              | [Mod loader](/modloader)/[Plugin loader](/pluginloader) | [Shader loader](/shaderloader) |
| ------------------------------ | ----------------------------------------------------------------- | --------------------------------- | ------------------------------------------------------- | ------------------------------ |
| [Resource pack](/resourcepack) | [Client](/client) ([Server](/resourcepack#server-resource-packs)) | [Optional](/resourcepack#version) | //                                                      | `Vanilla`                      |
| [Datapack](/datapack)          | [Client](/client) or/and [Server](/server)                        | Required                          | `Vanilla`                                               | //                             |
| [Shader](/shader)              | [Client](/client)                                                 | Required                          | //                                                      | Required                       |
| [Plugin](/plugin)              | [Server](/server)                                                 | Required                          | Required                                                | //                             |
| [Mod](/mod)                    | [Client](/client) or/and [Server](/server)                        | Required                          | Required                                                | //                             |
| [Modpack](/modpack)            | [Client](/client) or/and [Server](/server)                        | Required                          | Optional                                                | Optional                       |
