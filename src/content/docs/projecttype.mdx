Project types are all the addons that add, modify or extends [Minecraft](/minecraft) default features.

To understand each type is necessary to understand which part of the game they affect, where they run and the required software they need to use.

| Project type                   | [Gameplay](/gameplay)                                     | [Modding environment](/moddingenvironment)                        | [Version](/versions)              | [Loader](/loader)                                                    |
| ------------------------------ | --------------------------------------------------------- | ----------------------------------------------------------------- | --------------------------------- | -------------------------------------------------------------------- |
| [Resource pack](/resourcepack) | [Singleplayer](/singleplayer)                             | [Client](/client) ([Server](/resourcepack#server-resource-packs)) | [Optional](/resourcepack#version) | [Vanilla](/vanilla)                                                  |
| [Datapack](/datapack)          | [Singleplayer](/singleplayer)/[Multiplayer](/multiplayer) | [Client](/client) or [Server](/server)                            | Required                          | [Vanilla](/vanilla)                                                  |
| [Shader](/shader)              | [Singleplayer](/singleplayer)                             | [Client](/client)                                                 | Required                          | [Shader loader](/shaderloader)                                       |
| [Plugin](/plugin)              | [Singleplayer](/singleplayer)/[Multiplayer](/multiplayer) | [Server](/server)                                                 | Required                          | [Mod loader](/modloader)                                             |
| [Mod](/mod)                    | [Singleplayer](/singleplayer)/[Multiplayer](/multiplayer) | [Client](/client) or/and [Server](/server)                        | Required                          | [Mod loader](/modloader)                                             |
| [Modpack](/modpack)            | [Singleplayer](/singleplayer)/[Multiplayer](/multiplayer) | [Client](/client) or/and [Server](/server)                        | Required                          | Optional [Mod loader](/modloader) and [Shader loader](/shaderloader) |
