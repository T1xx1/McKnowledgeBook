Shader loaders are a type of [Loader](/loader) that introduce modifications to [Minecraft](/minecraft) rendering and texture system allowing [Shader](/shader) in the game, which dramatically improve the game's graphics by adding advanced lighting effects, shadows, reflections, custom textures and animations.

| Shader loader                                | Status |
| -------------------------------------------- | ------ |
| [OptiFine](https://optifine.net/home)        | Active |
| [Iris shaders](https://www.irisshaders.dev/) | Active |
| [Canvas](https://modrinth.com/mod/canvas)    | Beta   |
