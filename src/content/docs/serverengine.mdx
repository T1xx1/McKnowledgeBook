A server engine is a custom software intented to replace [Minecraft](/minecraft) [Vanilla](/vanilla) [Server file](/server).

Server engines rewrite the [Server file](/server) mostly for performance reasons.

| [Server](/server) engine                         | Language |
| ------------------------------------------------ | -------- |
| [Bamboo](https://github.com/macmv/bamboo)        | Rust     |
| [Feather](https://github.com/feather-rs/feather) | Rust     |
| [FerruMC](https://github.com/ferrumc-rs/ferrumc) | Rust     |
| [Glowstone](https://glowstone.net)               | Java     |
| [Hyperion](https://hyperion.rs)                  | Rust     |
| [KittyMC](https://github.com/Kek5chen/kittymc)   | Rust     |

| [PKMC](https://github.com/Vulae/pkmc)            | Rust     |
