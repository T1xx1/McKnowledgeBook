Target selectors are the `@` keywords used in [Commands](/command) to target specific players or entities.

| Scope                              | Entities                   | Players |
| ---------------------------------- | -------------------------- | ------- |
| All                                | `@e`                       | `@a`    |
| The nearest from command execution | `@n` <Edition type="je" /> | `@p`    |
| The one executing the command      | `@s`                       |         |
| Random                             |                            | `@r`    |
