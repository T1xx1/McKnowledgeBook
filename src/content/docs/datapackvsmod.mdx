|             | [Datapack](/datapack)                                                             | [Mod](/mod)                                                                                        |
| ----------- | --------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| Difficulty  | Easy                                                                              | Hard                                                                                               |
| Setup       | A code editor is recommended but not required                                     | A development environment with an IDE is necessary                                                 |
| Code        | `json`s and `png`s, and [Minecraft](/minecraft) [Commands](/command) | [Java](https://java.com)                                                                           |
| Multiplayer | Datapacks can be streamed from the [Server engine](/server)                       | Everyone has to pre-install required [Mods](/mod) from the [Server](/server) to be able to join    |
| Loader      | Datapacks run on the [Vanilla](/vanilla) engine                                   | [Mods](/mod) require [Mod loaders](/modloader)                                                     |
| Limitations | Limited possibilities defined by the [Vanilla](/vanilla) engine                   | "Infinite" possibilities                                                                           |
| Updates     | Performing updates it's easier                                                    | Updates are difficult to perform as [Mod loaders](/modloader) have to frequently change their APIs |
