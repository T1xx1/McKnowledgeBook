A launcher is a software application used to manage [Minecraft](/minecraft) installations and run the [Client JAR](/client).

It can be used manage your Minecraft account, customize player skins and <Wiki src="https://minecraft.wiki/w/Cape" name="Capes" />.

Apart from the official [Minecraft launcher](/minecraftlauncher), some other launchers are also able to install and run [Loaders](/loader) and non [Vanilla](/vanilla) [Project types](/projecttype).
