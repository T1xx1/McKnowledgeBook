A launcher is a software application used to manage and run [Minecraft](/minecraft) installations and its related content.
It also can be used manage the Minecraft account, customize player skins and <Wiki src="https://minecraft.wiki/w/Cape" name="Capes" />.

Apart from the official Minecraft launcher, other launchers are also able to install and run [Loaders](/loader) and [Project types](/projecttype).

- [Minecraft launcher](/minecraftlauncher) ([Mojang](/mojang) official launcher)
- [Modrinth App](/modrinthapp)
- [MultiMC](https://multimc.org)
- [Prism](https://prismlauncher.org)
- [XMLC](https://xmcl.vercel.app)
- [Lunar client](https://lunarclient.com)

<br />

:::note
Sometimes launchers are also referred to as clients from the [client-server architecture](/environment).
:::
