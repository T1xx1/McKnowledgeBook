Target selectors are specifiers used to select players or entities for [Commands](/commands).

| Selector                     | Scope                                     |
| ---------------------------- | ----------------------------------------- |
| `@a`                         | All players                               |
| `@p`                         | The nearest player from command execution |
| `@r`                         | A random player                           |
| `@e`                         | All entities                              |
| `@n` [JE](/javaedition) only | The nearest entity from command execution |
| `@s`                         | The entity executing the command          |
