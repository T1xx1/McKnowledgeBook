The datapack command is used to manage [Datapacks](/datapack) with [Commands](/command).

## Subcommands

| Command                                    | Description                                           |
| ------------------------------------------ | ----------------------------------------------------- |
| `datapack enable/disable <name>`           | Enable/disable a datapack                             |
| `datapack enable <name> (first \| last)`   | Enable a datapack as first/last                       |
| `datapack enable <name> (before \| after)` | Enable a datapack before/after another one            |
| `datapack list [available\|enabled]`       | List all datapacks or only the available/enabled ones |
| `datapack create <id> <description>`       | Create a new datapack in the current world            |
