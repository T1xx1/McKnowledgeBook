NeoForge is a [Mod loader](/modloader), forked from [Forge](/forge) due an internal dispute in the core team.

#### [Website](https://neoforged.net)

- [Discord](https://discord.neoforged.net)
- [GitHub](https://github.com/neoforged/NeoForge)

Technically [Mods](/mod) written for [Forge](/forge) can run on NeoForge too but that's not always true.
