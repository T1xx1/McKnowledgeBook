`/dialog` is a [Command](/command) introduced in <Version version="25w20a" /> that can display simple dialog interfaces to the player.

:::note
`/dialog` and `/dialogue` are two different commands. <Wiki slug="Commands/dialogue" name="Dialogue" />
:::

In [Singleplayer](/singleplayer) opening a dialog pauses the game like with the pause menu.
