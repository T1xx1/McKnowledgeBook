`/dialog` is a [Command](/command) introduced in the <Version version="25w20a" /> that can display simple dialog interfaces to the player defined in [Datapacks](/datapack).

:::note
`/dialog` and `/dialogue` are two different, non related with each other, commands. <Wiki slug="Commands/dialogue" name="Dialogue" />
:::

In [Singleplayer](/singleplayer) opening a dialog pauses the game like with the pause menu.
