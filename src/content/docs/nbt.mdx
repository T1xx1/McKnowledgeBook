The named binary tag format is a custom data tree format used by [Minecraft](/minecraft) to store and load data about blocks and entities.

For exmaple it's used to populate a [World](/world) with [Structures](/structure) upon [World generation](/worldgeneration).

[Structures](/structure) can be saved, loaded and edited with the [Structure block](/structureblock) and the [Jigsaw block](/jigsaw).

#### [SNBT](/snbt)

## Location

| [Launcher](/launcher)                    | Location                                                                      |
| ---------------------------------------- | ----------------------------------------------------------------------------- |
| [Minecraft launcher](/minecraftlauncher) | `.minecraft/saves/<world>/generated/minecraft/structures`                     |
| [ModrinthApp](/modrinthapp)              | `ModrinthApp/profiles/<profile>/saves/<world>/generated/minecraft/structures` |
