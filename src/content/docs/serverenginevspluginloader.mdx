Both [Server engines](/serverengine) and [Plugin loaders](/pluginloader) rewrite [Minecraft](/minecraft)'s [Vanilla](/vanilla) [Server code](/server). [Server engines](/serverengine) rewrite the [Server file](/server) mostly for performance reasons, [Plugin loaders](/pluginloader) rewrite the [Server file](/server) to implement [Plugins](/plugin).
