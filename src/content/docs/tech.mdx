- [Astro](https://astro.build) is a modern static site generator that allows you to build fast, content-focused websites using custom frameworks and libraries.
- [Starlight](https://starlight.astro.build) is a documentation theme for Astro that provides a clean and customizable layout for technical documentation.
- [MDX](https://mdxjs.com) is a markdown format that allows you to write JSX components within your markdown files, enabling you to create rich and interactive content.
- [Svelte](https://svelte.dev) is a lightweight and performant reactive framework for building user interfaces.
- [Tailwind CSS](https://tailwindcss.com) is a utility-first CSS framework that provides a set of pre-defined classes to help you build responsive and modern user interfaces quickly.
- [Vercel](https://vercel.com) is a cloud platform for static sites and serverless functions, providing fast and reliable hosting.
